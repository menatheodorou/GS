Overview

A connection type implementatino for mock devices

Tasks

Other Methods

  • – addMockDevice:

    Adds a mock device to the connection type. There are no mock devices available by default. You must add a mock device using this method to interact with it.
  • – removeMockDevice:

    Removes a mock device from the connection type.

Other Methods

  • – deviceType

    The type of device – this can be set to anything and retreived for printing or examining. It is not used by the framework.
